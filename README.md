## 🧠 System Architecture

This project simulates a real-world AI software engineering workflow using multiple agents:

- Planner Agent: decomposes requirements into tasks

- Coder Agent: generates executable code

- Reviewer Agent: analyzes code quality

- Fixer Agent: iteratively improves code

## 🔄 Execution Flow

User Input → Task Planning → Code Generation → Code Review → Auto Fix → Final Output
